MapMagic API Reference

Methods

  • /tweets/

    Returns tweets relevant to topics chosen in control panel (under development)

    Example:

    {
    body: "The flooding has hit the Charlotte area. #Florence",
    date: "Sun Sep 16 04:05:50 +0000 2018",
    id: 345,
    lat: 35.001706,
    lng: -81.046876,
    sentiment: "-0.352619",
    tweet_id: "1041176309261639681",
    weight: 1.6434777694708416
    }

  • /coords/

    Returns tweet coordinates and sentiment weight. Used for Twitter heatmap

    Example:

    {
    lat: 36.540739,
    lng: -83.67529,
    weight: 0.11688417068251102
    }

  • /users/

    Returns list of users. Used for the ResQ api

    Example:

    {
    id: 9403,
    name: "Bob Bobertson",
    age: 38,
    coords_home: {lat: 33.920239, lng: -51.22595},
    coords_last_seen: {lat: 33.894321, lng: -51.39219},
    date_registered: "Thu Sep 13 04:20:10 +0000 2018",
    phone: "2128675309",
    email: "bobertson@aol.com",
    supplies_requested: ["toilet paper", "first aid kit", "clean water"]
    }
